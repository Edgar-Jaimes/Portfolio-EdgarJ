My first cybersecurity competition experience focused on earning points by solving security challenges.

Project Description

This project highlights my first Capture-the-Flag (CTF) competition, which is a cybersecurity event where participants earn points by completing a variety of security challenges. The goal is to solve tasks as efficiently as possible and apply different cybersecurity concepts in a hands-on way.

During the competition, I worked through challenges focused on threat analysis, password cracking, and vulnerability exploitation. I used Kali Linux as my main environment to run tools, test solutions, and validate results. Each challenge required careful attention to detail and a step-by-step problem-solving approach.

I collaborated with teammates under time constraints, which pushed me to communicate clearly, divide tasks, and stay organized. This competition helped me improve how I approach technical problems when there is limited time and multiple challenges happening at once.

Skills Learned

  • CTF problem-solving: breaking down challenges into smaller steps to find solutions
  • Basic threat analysis: identifying what a challenge is asking and what evidence matters
  • Password cracking fundamentals and safe tool usage in a controlled environment
  • Introductory vulnerability exploitation concepts and validation
  • Using Kali Linux as a security testing environment
  • Teamwork, time management, and communication under pressure

Outcome

This CTF was my first competition experience and gave me a strong foundation in hands-on cybersecurity work. It strengthened my confidence with tools like Kali Linux and improved my ability to solve technical challenges quickly while collaborating with a team.
